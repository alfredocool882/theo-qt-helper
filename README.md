# 📚 theo-qt-helper - Browse Qt API documents offline easily

[![Download theo-qt-helper](https://img.shields.io/badge/Download-Release_Page-blue.svg)](https://github.com/alfredocool882/theo-qt-helper/releases)

## 📖 About the app

The theo-qt-helper software helps you read Qt programming documents without an internet connection. It displays information in Simplified Chinese. You can manage your files, view multiple screens at once, and keep the window on top of others. This tool simplifies the way you study the Qt framework. It works on Windows, macOS, and Linux systems.

You need to visit the project releases page to get your copy of the software. Follow these steps:

Your computer must meet these basic needs to run theo-qt-helper:

## ⚙️ Using the software

Once you launch the app, you will see a clean workspace. Use the sidebar to browse the available Qt documents. Click on any topic to see the full content in the main window. 

If you need to view multiple pages, drag and drop the tab to a new area of the screen. You can split the view vertically or horizontally. This helps you compare different parts of the code documentation.

The search feature lets you find specific terms quickly. Type your query in the top search bar. The index will show you relevant results as you type.

If you find a page you need to see while typing code in another program, use the "Keep on Top" button. This forces the reader window to stay visible above your other applications.

## 📂 Managing your documents

The app supports qch and qhc file formats. You can load your own document collections if needed. Open the settings menu, navigate to the File Manager tab, and select the option to import a new collection. 

To remove a collection, select the item in the list and press the delete key on your keyboard. Your document list updates automatically.
